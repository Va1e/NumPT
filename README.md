NumPT
=====

Numerik Aufgaben für Octave(MATLAB)

Die Plotaufgaben 1.2, 2.1, 4.1 muessen mit octave --persist geöffnet werden
damit man die plots sieht. sonst ist octave zu, bevor gnuplot fertig ist.
